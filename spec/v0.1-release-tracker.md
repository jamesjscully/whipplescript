# v0.1 — the real first release (tracker)

Status: **active**. Decision (Jack, 2026-07-16): the earlier v0.2/v0.3/v0.4
ladder was ceremony to establish a release cadence; there are no users, so
**collapse everything into a single polished, feature-complete, ready-to-use
`v0.1.0`.** Everything built ships (improve/evals included — no deferral). The
prior version history is discarded, not migrated.
